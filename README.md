## Running the Code

(Node.js version `v8.9.3`. npm verison `5.5.1`)

Clone the repo :
```
git clone https://github.com/SagarB-97/ColabIt.git
cd ColabIt
```

### Run the peer server

1. Install dependencies
```
cd peerserver
npm install
```

2. Run server
```
node index.js
```

### Run the coordinator

Run the following in the repository's root

1. Install dependencies :
```
npm install
```

2. Start the application :
```
npm start
```
